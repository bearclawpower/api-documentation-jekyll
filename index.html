Quick-start Instructions

1. Fill out the sign up form.

2. Receive an API access token via email.

3. Send requests with your access token in the Authorization header.

   $ curl -H "Authorization: Bearer Abc123XYZ" {{ site.api_protocol }}{{ site.api_url }}

4. PROFIT!
